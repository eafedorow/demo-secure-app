🧠 DOM XSS (Document Object Model XSS)

DOM XSS — это уязвимость, при которой вредоносный код попадает в DOM напрямую через JavaScript, без участия сервера.

📌 Примеры источников:

  • location.hash
  • location.search
  • localStorage
  • user input формы

⚠️ Пример опасного кода:

document.body.innerHTML = location.hash;

📚 Как работает DOM XSS

В отличие от обычного XSS, здесь сервер вообще не участвует. Уязвимость появляется из-за неправильной работы с DOM API.

💥 Как атакует злоумышленник:

  1. Передаёт payload через URL / input
  2. JS код вставляет его в DOM
  3. Браузер исполняет его как HTML/JS

🛡️ Как защититься:

  • Не использовать innerHTML с user input
  • Использовать Angular sanitizer
  • Использовать textContent вместо HTML
  • Фильтровать location.hash / search
